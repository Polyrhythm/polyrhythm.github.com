p

(p & s)
Make paragraph

    (p "hello")
    ;=> <p class="paragraph">hello</p>
    (p {:class "foo"} "hello")
    ;=> <p class="paragraph foo">hello</p>

Following markdown format is allowd.

 * Parse link
    "[hello](world)"
    ;=> <a href="world">hello</a>
    "[hello](title: POST TITLE)"
    ;=> <a href="POST URL">hello</a>

 * Parse emphasized
    "*hello*"
    ;=> <em>hello</em>

 * Parse strong
    "**hello**"
    ;=> <strong>hello</strong>

 * Parse inline code
    "`hello`"
    ;=> <code class="prettyprint">hello</code>

 * Parse new line
    "\n\nhello"
    ;=> <br />hello
